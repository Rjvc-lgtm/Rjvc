RJVC - Admin Panel

Welcome Admin

यह RJVC MLM सिस्टम का Admin Dashboard है।

अभी basic admin panel active है। अगले version में आप जो कहोगे वो जोड़ देंगे।

Admin देख सकता है:

  • कितने customers साइट का उपयोग कर रहे हैं
  • किस customer ने कितने member add किए
  • कौन किस level पर पहुँचा
  • हर customer की next level income
  • Daily joining रिपोर्ट
  • Income calculator की पूरी table
  • Withdrawal request approve / reject
  • Login करने वाले users का data

(यह basic version है। आप बताओ कि admin panel में क्या-क्या extra चाहिए — मैं पूरा जोड़ दूँगा।)
